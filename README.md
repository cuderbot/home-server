# Home server

Este repo es para mantener la configuración de los servicios
que corro en mis server.

# How to install

```
# clonar el repo
git clone https://github.com/cuderbot/home-server.git

# crear un archivo con variables de configuración
cp .env.example .env

# Levantar los docker
docker-compose up -d

# Listo!
```
